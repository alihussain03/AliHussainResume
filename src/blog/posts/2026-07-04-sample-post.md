This is a **sample post** showing the repeatable format for the blog. It exists only so the blog index has something to render — replace it with real writing.

## How to add a post

1. Create a Markdown file in `src/blog/posts/` named `YYYY-MM-DD-your-title.md`.
2. Add front matter at the top:

```yaml
---
title: "Your post title"
excerpt: "One or two sentences shown on the blog index."
---
```

3. Write the body in Markdown below the front matter. The date in the filename becomes the post date, and the rest of the filename becomes the URL slug.

That's it — the post appears on the [blog index](/blog/) automatically, newest first.
